# Squad Manager Tool - Gestão de Atletas e Logística

## 📖 Descrição
Este projeto simula um sistema de gestão para equipes de futsal, automatizando 
desde o cálculo de performance técnica dos atletas até o rateio de custos de viagem.

## 🛠️ Tecnologias Utilizadas
- JavaScript (ES6+)
- Node.js (Ambiente de execução)

## 🧠 Conceitos Aplicados
Nesta task, apliquei conceitos fundamentais de engenharia de software:
- **Manipulação de Arrays:** Uso de `push`, `pop`, `shift` e `unshift` para gestão de filas.
- **Programação Funcional:** Utilização de `map`, `filter` e `reduce` para processamento de dados imutáveis.
- **Lógica de Negócio:** Cálculos dinâmicos de média e rateio financeiro.

## Desafios
Um desafio superado foi garantir que a adição de novos membros ao array seguisse o contrato de dados (objeto com notas) para evitar erros de execução no método .map().

## 🚀 Como Executar
1. Clone o repositório.
2. Certifique-se de ter o Node.js instalado.
3. Execute o comando: `node src/app.js`
